Project Planning / Development

Throughout my college admissions process, I often found it difficult to access clear, organized information about schools, deadlines, and requirements. I decided to make a website that provides students with valuable resources to confidently navigate their admissions journey. Some challenges I faced during this was creating animations and making an interactive map for the users. I used the Leaflet API for the interactive map, which made it so when the user types in their address, it runs through the system and pops up on the map, and same with the college name. For the animations I used complex online resources. I also used a spreadsheet to stay organized with all the data I web-scraped and pulled using APIs for the thousands of schools.
