Project Planning / Mentorship

I had struggles finding colleges as I was going through the admissions process so I decided to make a website so it is easier for other students. At first I had issues with finding an API to use for tuition costs because of limited data, but I found a new source that I will implement soon. I used AI to pull some info and used leaflet for the map. My mentor helped me through providing me resources to refine my project such as spreadsheets, APIs, and videos.
